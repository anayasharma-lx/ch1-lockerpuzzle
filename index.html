The Locker Puzzle

100 lockers, 100 students. Student n toggles every nth locker. Which lockers stay open? Run the simulation — then click any tile to find out why.

The key insight: A locker ends up open only if it is toggled an odd number of times. That happens exactly when the locker number has an odd number of factors — which is only true for perfect squares.
