Key insight
A locker stays open only if toggled an odd number of times. That happens only when its number is a perfect square — because only squares have an odd number of factors.

Why only squares?
Every factor d of n pairs with n/d. Pairs cancel (toggle twice = closed). Only when d = n/d — i.e. d = √n — does a factor stand alone, giving an odd count of toggles and leaving the locker open.
